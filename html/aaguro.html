AAGURO: Autonomous Line Follower Robot

AAGURO is an autonomous line follower robot designed using an Arduino Nano microcontroller. The primary objective of this project is to create a robot capable of transporting objects from one point to another. To achieve this, the robot is equipped with an electromagnetic sensor at the rear, enabling it to attract and pull objects along its path.





Skills: Electronics,Embedded System

Contribution:

  • Choosing and make the electronic schematic for the robot's operation.
  • Soldering and wiring the electronic components
  • Ensuring all electronic components work seamlessly together, from the power supply to the sensors and actuators.
