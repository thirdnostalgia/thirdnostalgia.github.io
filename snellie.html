Snellie: Arduino Based Forklift Robot

This project was created to complete a robotics experimental II class assignment. The main task of this robot is to lift objects and place them in a predetermined position.

The design of this robot looks like a snail so we call it Snellie. The body is made of acrylic and PLA filament which is made using a 3D Printer. The wheels use mecanum so they can be maneuvered in all directions more easily. For the controller, this robot uses an Arduino nano which controls the DC motor for the robot's movement, as well as controlling the servo to rotate and raise the fork to lift objects. This robot is controlled using a smartphone via Bluetooth.

Skills: 3D Design, CAD, 3D Print, Electronics, C++, Arduino
